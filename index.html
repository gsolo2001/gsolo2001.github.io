Uno game

To research and better understand the capabilities of a Cyclic Doubly Linked List (CDLL), I worked with a small team to create a digital game of Uno that used a CDLL to manage and sort the deck of cards used as well as the players involved in the game.

See also Media Player application.

Media player

In addition to an Uno game, my team and I also created a media player to delve into the capabilities of a Cyclic Doubly Linked List (CDLL). The media player uses the CDLL to manage, sort, and repeat the song queue.

See also Sonic Media Player presentation.

Vending machine system

To practice and improve our agile software development skills, I worked with a small team to design and develop a software that would allow you to manage a network of vending machines. We coded the project in Java using Eclipse IDE and focused on back-end features such as data permanence and proper code structure. Throughout the project, we used GitHub for version control and also to track tasks and organize our sprints.

Supernova

For my capstone project, I worked with a group to create a 2D, retro-style game using all the knowledge I'd gathered so far. We used Microsoft Visual Studio to design and devlop the game from scratch and GitHub for version control and to organize our sprints.

We coded the game from the ground-up using C++ and designed all of its assets ourselves. The custom-built game engine included physics and rendering, and all the game code was written using object-oriented programming. The graphics, artwork, and sound effects were all custom designed and created by our talented team.

View the Project on GitHub

Ivanti neurons redesign

As part of my role as System Administrator for Valdosta State University's work order system (run by Ivanti Neurons for ITSM), I led and executed a project to redesign the end-user experience. The goal was to simplify and beautify the user interface and make it more functional. To ensure user satisfaction and adoption, I additionally organized and led focus groups to introduce the new design and features.
